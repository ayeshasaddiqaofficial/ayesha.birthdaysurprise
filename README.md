## 🚀 How to Use

### Just Open It
```
Double-click birthday.html → opens in browser → done!
```

### Host It Online (Free)
1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag and drop `birthday.html`
3. Get a shareable link — send it! 💌

### Customize It

**Change his name:**
Find this line near the top of the `<script>` section and replace:
```javascript
const HUSBAND_NAME = "My Love"; // Change this!
```

**Replace photos:**
1. Upload your photos to [imgbb.com](https://imgbb.com)
2. Copy the "Direct link"
3. Replace the `picsum.photos` URLs in the timeline section

**Edit the love letter:**
Find the "A Letter For You" section in the HTML and change the text.
